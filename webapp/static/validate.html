Does the quick probe (used by Tune & Model Eval) predict what the real agent actually does? We ask the probe which tool it would pick, then run the real agent and watch which tools it actually calls. Predicts first tool = probe matched the agent's first call (exact). Right tool used = the probe's tool showed up somewhere in the agent's run (fairer for multi-step questions, where the agent gathers basics first).
